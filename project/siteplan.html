Dan Code Camp — Site Plan

Site Name

Dan Code Camp

The name was chosen because the website focuses on teaching coding skills in a camp-style learning format. It is simple, friendly, and easy to remember.

Optional domain: dancodecamp.com

Site Purpose

Dan Code Camp is an educational website that introduces beginners to coding. It provides lessons, beginner guides, and coding resources to help users start programming with confidence.

Scenarios

  • Where should I start if I am completely new to coding?
  • Which programming language should I learn first?

Color Schema

These colors will be used across the website:

Deep Blue (#1f3c88) — headers and navigation
Orange (#f39c12) — highlights and accents

Typography

JetBrains Mono — headings and code-style text

Roboto — body text and paragraphs
